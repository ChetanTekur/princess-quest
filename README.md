# Princess Quest: The Castle Escape

A retro-style educational puzzle game for kids (approx. age 5). Help the Princess escape the castle by solving math, spelling, and logic puzzles!

This game is built as a **Single File Application**.

1.  Locate the `index.html` file in this folder.
2.  **Double-click** it.
3.  The game will open in your default web browser. No internet or server required!

## üåç How to Publish on GitHub (Share with Friends)

You can host this game for free using **GitHub Pages**.

1.  **Create a GitHub Account** at [github.com](https://github.com).
2.  **Create a New Repository**:
    *   Click the **+** icon in the top right -> **New repository**.
    *   Name it `princess-quest`.
    *   Check **"Add a README file"**.
    *   Click **Create repository**.
3.  **Upload the File**:
    *   Click **Add file** -> **Upload files**.
    *   Drag and drop your `index.html` file here.
    *   Click **Commit changes**.
4.  **Enable GitHub Pages**:
    *   Go to **Settings** (tab at the top of your repo).
    *   Scroll down (or click "Pages" on the left sidebar) to the **GitHub Pages** section.
    *   Under **Source**, select `main` (or `master`) branch and `/ (root)` folder.
    *   Click **Save**.
5.  **Play!**
    *   Wait about 1-2 minutes.
    *   GitHub will give you a link (usually `https://your-username.github.io/princess-quest/`).
    *   Send this link to anyone!
